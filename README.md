## Executando o projeto 
### Ajustando o .env
  Renomeie ``.env.example`` para ``.env`` e configure como desejar.
  
### Executando a postgres

```bash
$ docker-compose up
```

### Instalando as dependências
  
```bash
$ npm install
```

### Executando a aplicação

```bash
$ yarn start:dev
```
